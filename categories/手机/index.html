小米手机解锁 Bootloader

Bootloader，简称BL，其手机安全启动规范的底层的一环。主要作用是验证加载对象签名，防止启动第三方 ROM 或 Recovery，避免由此带来的数据安全风险。同时限制了 Fastboot 方式刷机（线刷），防范手机在没有授权的情况下被刷机。

解锁Bootloader，仅适合高级玩家，小白切勿自行尝试。此外，根据解锁规则，解锁BL后因刷机导致的硬件故障，小米公司不予保修。

解锁Bootloader会清除手机数据，有重要数据请备份再操作！！！

Surfboard 使用教程

Surfboard，直译为冲浪板，可能是 Android 安卓系统平台最快速的科学上网代理工具。Surfboard 兼容 Surge 配置，可以将 Surge标准的配置文件导入直接使用。

Surfboard 使用 VpnService API 拦截系统流量进行转发/修改/屏蔽等处理。

目前已经支持 HTTP / HTTPS / SOCK5 / SOCKS5 over TLS / SS/SS-OBFS / Vmess/Trojan 等协议。不过，官方明确表示不会支持 SSR 协议。

ROOT 环境隐藏

Shamiko是一个与Magisk一起使用的模块，它可以在开启Zygisk的情况下隐藏root权限。这对于需要在root过的设备上使用那些通常不允许root权限的应用（如银行、金融和游戏类应用）非常有用。

Shamiko模块可以隐藏Magisk、Zygisk本身、二进制文件”su”，甚至在某些设备上隐藏”Bootloader未锁定”的状态。安装Shamiko后，默认是黑名单模式，这意味着你需要使用Magisk的排除列表来对特定应用隐藏root。

LSPosed 安装教程

Xposed 框架是一款可以在不修改 APK的情况下影响程序运行（修改系统）的框架服务，基于它可以制作出许多功能强大的模块，且在功能不冲突的情况下同时运作。

LSPosed 是 Edxposed 的一个新的分支，LSPosed 与Edxposed 相互兼容，却比 Edxposed 更加简洁、易用和高效，是现如今饱受诟病的 Edxposed 的极佳替代品。

与 Xposed 相同，LPSosed 可以在不真实修改任何应用和系统组件的情况下达到修改的目的，从而实现强大的功能。

2024年1月8日晚间时候， LSPosed 项目团队宣布停止 LSPosed 模块管理软件项目更新及维护工作。
