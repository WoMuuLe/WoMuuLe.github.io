LSPosed 安装教程

Xposed 框架 是一款能够在 不修改 APK 文件 的情况下影响程序运行（即修改系统行为）的强大框架服务。基于此框架，可以开发出众多 功能强大 的模块，并且在功能不冲突的前提下，这些模块可以同时运作。

LSPosed 是 Edxposed 的一个新分支，LSPosed 不仅与 Edxposed 相互兼容，而且相比 Edxposed 更加 简洁、易用和高效，成为了当前备受争议的 Edxposed 的绝佳 替代方案。

与 Xposed 类似，LSPosed 可以在 不实际修改任何应用和系统组件 的情况下达到修改目的，从而实现强大的功能。

2024年1月8日晚，LSPosed 项目团队宣布停止 LSPosed 模块管理软件项目的更新及维护工作。

前提准备

  • 安装 Magisk，详见 Magisk 安装教程。

步骤演示

1. 模块下载

下载 LSPosed 模块（Zygisk 版）：

  • 蓝奏盘
  • GitHub

部分地区可能无法访问 GitHub，可以选择从蓝奏盘下载。

2. 开启 Zygisk

  • 打开 Magisk – 设置 – 开启 Zygisk

3. 模块安装

  • 打开 Magisk – 模块 – 从本地安装

4. 进入软件

  • 重启设备后，通过通知栏进入 LSPosed（如果没有显示，可以通过拨号键输入 ***#\*#5776733#\*#\*** 进入 LSPosed）

5. 基本设置

  • 创建快捷方式
  • 关闭状态通知

6. 完成安装

  • LSPosed 显示“已激活”即表示安装成功

请注意，上述操作涉及系统级修改，有一定的风险性和复杂度。建议在操作前确保了解每一步骤，并做好相应的备份措施。

ROOT 环境隐藏

Shamiko是一个与Magisk一起使用的模块，它可以在开启Zygisk的情况下隐藏root权限。这对于需要在root过的设备上使用那些通常不允许root权限的应用（如银行、金融和游戏类应用）非常有用。

Shamiko模块可以隐藏Magisk、Zygisk本身、二进制文件”su”，甚至在某些设备上隐藏”Bootloader未锁定”的状态。安装Shamiko后，默认是黑名单模式，这意味着你需要使用Magisk的排除列表来对特定应用隐藏root。

Magisk 安装教程

Magisk 是一套开源的 Android（支持6.0及以上版本）自定义工具集，其中包括 Magisk Manager（图形化管理界面）、ROOT 功能、启动脚本、SElinux 补丁 以及 启动时认证/dm-verity/强制加密移除 等功能。Magisk 提供了一种无需直接修改系统文件即可 更改 /system 或 /vendor 分区内容 的方法，并通过类似 Xposed 的模块系统，允许开发者对系统进行修改或增强所安装应用的功能。

对于 内核版本为 Android 13-14 的设备，请参考特定教程：内核安卓13-14刷面具（修补init_boot.img）

演示环境

  • 手机型号：小米 10 Pro
  • 系统版本：MIUI 14（基于 Android 13）

前提准备

  • 已解锁Bootloader，详情见 《小米手机解锁Bootloader》 教程。

演示步骤

  1. 下载系统包

    • 从 XiaomiROM 下载与手机系统版本一致的系统包。

  2. 提取 boot.img 文件

    • 详见 payload-dumper-go 提取 boot。（如果系统包解压后直接包含 boot.img 文件，则可跳过此步）

  3. 传输文件至手机

    • 将 boot.img 文件通过 USB 数据传输模式拷贝到手机的 Download 文件夹。

  4. 安装 Magisk 应用

    • 从 Magisk 下载并安装 Magisk 应用。

  5. 修补 boot.img 文件

    • 打开 Magisk 应用，依次选择“安装”->“选择 boot.img”->“开始修补”->“修补成功”。

  6. 下载 adb-fastboot 工具

    • 从 蓝奏盘 下载 adb-fastboot 工具包并解压。

  7. 传输修补后的文件

    • 通过 USB 将修补后的 boot.img 和 magisk.img 文件拷贝到电脑的 adb-fastboot 工具目录。

  8. 进入 Bootloader 模式

    • 关闭手机，同时长按电源键和音量减键，进入 Bootloader 模式后连接至电脑。

  9. 使用 Fastboot 刷入修补后的文件

小米手机解锁 Bootloader

Bootloader（简称BL）是手机安全启动规范中的一个重要组成部分。它的主要功能包括验证加载对象的数字签名，以防止未经授权的第三方ROM或Recovery被启动，从而保护用户的隐私和数据安全。此外，它还限制了使用Fastboot方式刷写固件，以防止未经授权的刷机操作。

解锁Bootloader是一项高级操作，仅推荐给熟悉此流程并了解潜在风险的用户。对于初学者来说，这项操作可能会带来不可预知的问题，因此不建议自行尝试。请注意，解锁后，由于刷机而导致的任何硬件故障将不再享受小米公司的保修服务。

解锁Bootloader会导致所有用户数据被清除，因此，在开始此过程之前，请确保已经备份了所有重要的个人数据。
