Clash Verge 使用教程

Clash Verge是代理工具Clash内核的GUI图形客户端，支持Windows、Linux、macOS系统，分流规则功能强大且支持多种代理协议，如V2Ray、Trojan、Shadowsocks、ShadowsocksR、Socks等协议，使用方法和功能和Clash for Windows类似，可以完美代替Clash for Windows使用。

下载和安装

点击这里下载 Clash Verge。Windows 用户可选择下载 .msi 或 .exe 安装文件，Mac 用户根据自己的芯片型号（Intel 或 M1/M2）下载相应的 .dmg 文件根据操作系统从下载对应平台的安装包进行安装。

按照提示完成安装。安装完成后，可在设置中将界面语言切换为中文。

PS: Clash Verge 只是客户端，需要配合代理节点一起使用。

机场：魔界.net、WindyCloud

获取订阅地址

进入机场，购买合适的订阅套餐，复制订阅地址。

配置代理

在 Clash Verge 中配置代理，首先在订阅界面填入订阅地址，然后导入代理配置文件。

如果是自建节点服务器，需要自行编辑配置文件并导入，注意 Clash Verge 支持的协议包括：Shadowsocks(SS)、ShadowsocksR(SSR)、Socks、Snell、V2Ray、Trojan 等。

选择代理模式

进入代理页面选择模式：

  • 规则模式（Rule）：根据配置文件的规则进行分流，只有需要代理的网站才会经过代理服务器。（推荐）
  • 全局模式（Global）：所有请求直接通过代理服务器。
  • 直连模式（Direct）：所有请求直接发送到目的地，相当于没开代理。

选择代理节点

根据被墙网站服务器所在地区或延迟自行选择代理节点。

开启代理

进入设置界面，打开系统代理选项，即可进行科学上网了。

注意：关机前务必关闭系统代理选项，否则下次开机后无法上网。解决方法：重新打开软件即可。

开始使用

启动系统代理后，您就可以开始浏览或使用被墙的网站和服务了。首次使用时建议在隐私模式下打开Google或YouTube测试，以避免浏览器插件影响代理的正常使用。

其他设置与建议

  • 语言切换：在设置中可将 Clash Verge 切换为中文界面。
  • 内核切换：可在设置中切换至 Clash Meta 内核，以支持更多类型的节点。
  • 导入机场订阅地址：直接从机场后台复制订阅地址，然后在 Clash Verge 中导入。
  • 默认端口：Clash Verge 的默认端口与 Clash for Windows 相同，为 7890，适用于如 Telegram 等应用的代理设置。
  • Tun 模式：开启后会生成虚拟网卡，使所有流量都通过代理。
