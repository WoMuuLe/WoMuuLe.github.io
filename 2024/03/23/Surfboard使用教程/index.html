Surfboard 使用教程

Surfboard，直译为冲浪板，可能是 Android 安卓系统平台最快速的科学上网代理工具。Surfboard 兼容 Surge 配置，可以将 Surge标准的配置文件导入直接使用。

Surfboard 使用 VpnService API 拦截系统流量进行转发/修改/屏蔽等处理。

目前已经支持 HTTP / HTTPS / SOCK5 / SOCKS5 over TLS / SS/SS-OBFS / Vmess/Trojan 等协议。不过，官方明确表示不会支持 SSR 协议。

下载和安装

点击这里下载 Surfboard。

PS: Surfboard 只是手机客户端，需要配合代理节点一起使用。

机场：魔界.net、WindyCloud

获取订阅地址

进入机场，购买合适的订阅套餐，直接导入或复制订阅地址。

配置代理

如需手动导入配置，在配置界面点击加号填入订阅地址，导入代理配置文件。

在仪表盘页面选择代理模式：

  • 规则模式（Rule）：根据配置文件的规则进行分流，只有需要代理的网站才会经过代理服务器。（推荐）
  • 直连模式（Direct）：所有请求直接发送到目的地，相当于没开代理。
  • 全局模式（Global）：所有请求直接通过代理服务器。

在代理界面根据被墙网站服务器所在地区或延迟自行选择代理节点。

开启代理

在仪表盘界面，点击右下角按钮即可启动系统代理。

开始使用

启动系统代理后，您就可以开始浏览或使用被墙的网站和服务了。首次使用时建议在隐私模式下打开Google或YouTube测试，以避免浏览器插件影响代理的正常使用。
